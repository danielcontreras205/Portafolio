Sistema de información Farmaser

El sistema de información de Farmaser fue un proyecto que se realizó en el transcurso que hacia mi carrera profesional. Este proyecto tenía como finalidad la construcción de un sistema de administración y gestión farmacéutica funcional y que tuviera la capacidad de satisfacer las necesidades expuestas por el cliente.


En cuanto al almacenamiento y consulta de la información se generaba internamente. El sistema estaba compuesto por la aplicación principal, que era totalmente de escritorio e interactiva con el usuario final. La aplicación de escritorio tenia que ser desplegada en un portátil que utilizaba la farmacia y en la cual se suministraba y monitoree la información.
1. Logi

Interface del login del aplicativo

En la captura visual proporcionada, se presenta la interfaz de inicio de sesión de la aplicación de Famaser. En este proceso, la plataforma realiza una validación interna que verifica la autenticidad del usuario mediante la introducción de un nombre de usuario y contraseña. Es importante destacar que dichos datos deben estar previamente registrados en la base de datos del sistema. Además, se destaca que las contraseñas y nombres de usuario se encuentran cifrados para garantizar la seguridad y confidencialidad de la información del usuario.
2. Lobby del aplicativo

Interface Principal

En la imagen proporcionada, se visualiza el lobby o panel principal de la aplicación de Famaser. Este espacio central congrega diversos módulos y funciones destinadas a la gestión integral del sistema. Los usuarios de la aplicación tienen la capacidad de llevar a cabo acciones como el registro, actualización y modificación de información relacionada con distintos tipos de medicamentos, usuarios y proveedores. Este diseño modular y multifuncional demuestra la versatilidad del aplicativo, permitiendo a los usuarios realizar tareas clave para la administración eficiente de datos en el ámbito farmacéutico.
